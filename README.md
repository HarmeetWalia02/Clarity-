🚀 Clarity
Adaptive AI Learning Platform for Inclusive Education

Transforming PDFs & Meetings into Personalized Learning Experiences

🌍 About Clarity

Clarity is an AI-powered adaptive learning platform built to make academic content accessible for every learner — including neurodiverse students, multilingual users, and learners with vision or hearing challenges.

It converts static documents and live meetings into structured, simplified, translated, spoken, and assessable content — all in one intelligent system.

✨ Key Features
📚 AI PDF Intelligence

Upload any academic PDF and instantly get:

✅ Clean structured version
📌 Bullet summary
🧠 Key concepts
📖 Definitions
📝 Exam-style questions
🎯 AI-generated quiz
🧠 Adaptive Learning Engine

Choose your learning mode:

Mode	Purpose
🎯 Focus Mode	Sentence-by-sentence highlight reading
📘 Simplified Reading	Easy-to-understand explanations
🔊 Audio Learning	Text-to-speech playback
🌍 Multilingual Mode	Translate + Voice output
📝 Assessment Mode	Instant AI quiz generation
🎤 Meeting Assistant
🎙 Live speech-to-text transcription
🗒 AI-structured meeting notes
♿ Inclusive for hearing-impaired users
📄 Save notes for revision
🎯 Accessibility Impact

Clarity adapts to different learner needs:
🧠 ADHD → Focus Mode
📖 Dyslexia → Simplified Reading
👁 Vision Support → Large UI + Audio
🌍 Language Barrier → Multilingual Translation
🎓 Exam Prep → AI Quiz Engine
🦻 Hearing Support → Live Captions

Education should adapt to students — not the other way around.

🚀 Getting Started
1️⃣ Clone Repository
git clone https://github.com/HarmeetWalia02/Clarity-.git
cd Clarity-

2️⃣ Install Dependencies
npm install

3️⃣ Add Environment Variable
Create .env.local:
GROQ_API_KEY=your_api_key_here

4️⃣ Run Development Server
npm run dev

Visit:http://localhost:3000
